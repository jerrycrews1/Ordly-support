# Ordly Support

Ordly is a local-only habit tracker for iPhone.

## Contact
Email: jerry@nerdquad.com

## Common Questions

### Where is my data stored?
All data is stored locally on your device (v1.0). No account required.

### How do I export my data?
Open Ordly → Settings → Export → choose CSV or JSON.

### I found a bug. What should I include?
- iPhone model
- iOS version
- What you did right before the issue
- Screenshots if possible
